# CityInfo

Приложение для просмотра информации о городах мира.

## 📱 Стек технологий

| Технология | Назначение |
|------------|------------|
| **Kotlin** | Язык программирования |
| **Jetpack Compose** | UI фреймворк |
| **Orbit MVI** | Управление состоянием (MVI) |
| **Ktor** | HTTP клиент |
| **Hilt** | Внедрение зависимостей |
| **Paging 3** | Пагинация |
| **Navigation 3** | Навигация |
| **Clean Architecture** | Архитектура приложения |

---

+ ## 🏗️ Архитектура
+ Проект построен на **Clean Architecture**:

+ app/          → Главный модуль 
+ core/         → Общие утилиты, тема, навигация
+ domain/       → Бизнес-логика (UseCase, модели, интерфейсы репозиториев)
+ data/         → Реализация репозиториев, API
+ feature/      → Фичи (экраны)
+   ├── cities/       → Список городов
+   └── citydetails/  → Детали города
+ `


## 📋 Функционал

### Экран списка городов
- 🔍 Поиск городов в реальном времени
- 📄 Бесконечная пагинация (Paging 3)
- ⏳ Шиммеры при загрузке
- 📌 Иконка + название города + страна
- ❌ Обработка ошибок сети

### Экран деталей города
- 📋 Информация о городе (название, страна, население)
- 🌐 Кнопка "Поиск информации о городе" (открывает браузер)
- ⬅️ Возврат к списку
